# Interior Decoration Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Decorate all village interiors with reusable prop art, selective furniture collision, ambient people, and short villager dialogue.

**Architecture:** Keep all authored room data in `src/lib/game/content/maps.ts` and keep `WorldScene` as the only exploration renderer. Add one interior prop asset manifest and one static PNG sheet, render props and ambient NPCs from map data, and route new interactable villager dialogue through the Rust story catalog plus browser fixture.

**Tech Stack:** TypeScript, Phaser 4, Svelte 5/Vite, Bun, Vitest, Cargo story checker, generated transparent PNG assets.

Add these helper functions after `isPointInsideRect`:

```ts
function expectRectClearOfRect(rect: CenterRect, blocker: CenterRect, message: string) {
	const overlaps =
		rect.x - rect.width / 2 < blocker.x + blocker.width / 2 &&
		rect.x + rect.width / 2 > blocker.x - blocker.width / 2 &&
		rect.y - rect.height / 2 < blocker.y + blocker.height / 2 &&
		rect.y + rect.height / 2 > blocker.y - blocker.height / 2;

	expect(overlaps, message).toBe(false);
}

function expectPointClearOfInteriorPropCollisions(
	map: WorldMapDefinition,
	point: { x: number; y: number },
	label: string
) {
	const pointRect = { x: point.x, y: point.y, width: 24, height: 24 };

	for (const prop of map.interiorProps ?? []) {
		if (!prop.collision) {
			continue;
		}

		expectRectClearOfRect(pointRect, prop.collision, `${map.id}:${label} blocked by ${prop.id}`);
	}
}
```
